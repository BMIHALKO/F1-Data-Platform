# F1-Data-Platform

Schedule-driven Formula 1 analytics platform built with Python, Snowflake, Airflow, Docker, and dashboard visualization.

## Project Scope

This project follows a phased development roadmap:

1. Foundation and architecture
2. API source exploration
3. Snowflake schema design
4. Ingestion pipeline
5. Transformations and analytics views
6. Dockerization
7. Airflow orchestration
8. Race-schedule refresh logic
9. Dashboard layer
10. Polish and expansion
